# 声纹指纹门锁系统 — 完整方案 (Arduino Nano ESP32 版)

> **MCU**: Arduino Nano ESP32 (NORA-W106-10B / ESP32-S3, 3.3V I/O, 原生 USB-CDC)
> **特点**: 全引脚 PWM、硬件串口 Serial1、3.3V 与指纹模块电平兼容、无需软串口
> **传感器**: TM1026M 指纹、KY-038 麦克风、MAX9812 声音波形、SSD1306 OLED、ISD1820 录音、5V 继电器、共阴 RGB、无源蜂鸣器

---

## 一、系统工作流程

```
待机 (OLED 锁屏 + 红色呼吸 + 滚动横幅)
   │  KY-038 / MAX9812 检测到声音
   ▼
唤醒动画 (麦克风 + 声波扩散) + RGB 蓝 + "叮咚"
   │
   ▼
等待指纹 (扫描动画 + 双向流动进度条 + 蓝色脉冲)
   │
   ├─ 匹配成功 ──► 开锁动画(放射线 + 烟花 + WELCOME)
   │              RGB 绿色渐暗 + 继电器吸合
   │              成功旋律 (每 5 次解锁触发"超级玛丽"彩蛋)
   │              ISD1820 播放"欢迎回家"
   │              5 秒后自动上锁
   │
   └─ 匹配失败 ──► 抖动 X 动画 + 警告音 + RGB 红
```

---

## 二、引脚分配表 (Nano ESP32)

| Nano ESP32 引脚 | 连接组件 | 说明 |
|:---:|:---:|:---|
| **D0 / RX0** | 指纹 TX (绿) | `Serial1` 硬件串口 RX |
| **D1 / TX0** | 指纹 RX (白) | `Serial1` 硬件串口 TX |
| **D2** | ISD1820 PLAYE | 边沿触发播放 |
| **D3** | 蜂鸣器 S | PWM 输出音调 |
| **D4** | 继电器 IN | 控制电磁锁 |
| **D5** | RGB-R | PWM (220Ω 限流) |
| **D6** | RGB-G | PWM (220Ω 限流) |
| **D7** | RGB-B | PWM (220Ω 限流) |
| **D8** | KY-038 DO | 数字声音触发 |
| **A0** | MAX9812 OUT | 模拟波形 (10-bit ADC) |
| **A4 / SDA** | OLED SDA | I²C 数据 |
| **A5 / SCL** | OLED SCL | I²C 时钟 |
| **3V3** | 指纹/MAX9812/OLED | 3.3V 模块供电 |
| **5V (VBUS)** | KY-038/继电器/ISD1820 | 5V 模块供电 (USB 或 VIN) |
| **GND** | 全部模块共地 | **必须共地** |

> ⚠ **重要差异**:
> 1. Nano ESP32 是 **3.3V 逻辑**, 与 5V Nano 不同, 不要把模块的 5V 输出反灌到数据脚。
> 2. `Serial` 是 USB-CDC, 不占用 D0/D1, 串口监视器与 Serial1 互不干扰。
> 3. 所有 D0–D12 都能 `analogWrite()` 输出 PWM, RGB 可真彩呼吸。

## 三、详细接线图

### 1. OLED 0.96" (I²C / SSD1306)

```
OLED       Nano ESP32
────       ───────────
VCC  ───►  3V3
GND  ───►  GND
SDA  ───►  A4
SCL  ───►  A5
```
默认地址 `0x3C`, 部分模块为 `0x3D` (代码已自动尝试两个)。

### 2. 指纹模块 TM1026M (6Pin, UART)

```
TM1026M           Nano ESP32
─────────         ───────────
VCC (红)   ─────► 3V3   (TM1026M 工作 3.3V, 切勿接 5V!)
GND (黑)   ─────► GND
TX  (绿)   ─────► D0 (RX0 / Serial1-RX)
RX  (白)   ─────► D1 (TX0 / Serial1-TX)
TOUCH/IRQ  ─────► (可不接)
3V3-OUT    ─────► (悬空)
```

> ✅ **3.3V 直连无需分压电阻**; 默认波特率 57600, 与 AS608 协议兼容, 直接用 `Adafruit_Fingerprint`。

### 3. 无源蜂鸣器

```
Buzzer       Nano ESP32
──────       ───────────
S(+)   ────► D3 (PWM)
VCC    ────► 5V         (有源板使用)
GND    ────► GND
```
两脚版: 正→D3, 负→GND 即可。

### 4. 1 路继电器 (5V)

```
继电器 (输入)        Nano ESP32
────────────        ───────────
VCC   ────────────► 5V
GND   ────────────► GND
IN    ────────────► D4

继电器 (输出)        电磁锁回路
────────────        ─────────
COM   ────────────► 12V 适配器 (+)
NO    ────────────► 电磁锁 (+)
电磁锁 (-) ────────► 12V 适配器 (-)
```

```
续流二极管保护:
   电磁锁(+) ──┬── 1N4007 (负极朝 +) ──┬── 电磁锁(-)
               │                       │
               └───────────────────────┘
```

### 5. ISD1820 录音模块 + 喇叭

```
ISD1820        Nano ESP32
───────        ───────────
VCC      ────► 5V (或 3V3, 看模块标注)
GND      ────► GND
PLAYE    ────► D2          (高脉冲 → 完整播放一次)
REC      ────► (录音时手动按住板载按键)
SP+ / SP- ──► 接 8Ω 小喇叭
```

### 6. KY-038 数字声音检测

```
KY-038       Nano ESP32
──────       ───────────
+ (VCC) ───► 5V
G (GND) ───► GND
DO      ───► D8       (检测到声音 = LOW)
AO      ───► (不接, 由 MAX9812 提供模拟波形)
```
板载电位器先转到中间, 调试时再细调。

### 7. MAX9812 声音波形

```
MAX9812      Nano ESP32
───────      ───────────
VCC   ─────► 3V3
GND   ─────► GND
OUT   ─────► A0
```

### 8. RGB LED (共阴极, 默认)

```
RGB              Nano ESP32
───              ───────────
R ──[220Ω]────►  D5 (PWM)
G ──[220Ω]────►  D6 (PWM)
B ──[220Ω]────►  D7 (PWM)
GND ──────────►  GND
```
共阳极: 公共脚接 3V3, 并把代码里 `RGB_COMMON_ANODE` 设为 `true`。

### 9. 2.2" 串口触摸屏

> 本方案不使用 (引脚优先 OLED), 后续升级可替换。

## 四、供电方案

```
[12V/2A 适配器] ──┬─► 12V 电磁锁 (经继电器)
                  │
                  └─► [12V→5V 降压] ──► Nano ESP32 VIN(or USB 5V)
                                         └─► 所有 5V 模块 (KY-038/继电器/ISD1820)

Nano ESP32 板载 LDO ──► 3V3 ──► OLED / TM1026M / MAX9812
```

> **共地**是稳定运行的关键。USB 调试时也要把外部 12V 系统的 GND 与 USB GND 连起来。

## 六、操作步骤

### Step 1: Arduino IDE 准备

1. **IDE 1.8.19+ 或 IDE 2.x**, `文件 → 首选项 → 附加开发板管理网址`:
   ```
   https://downloads.arduino.cc/packages/package_index.json
   ```
2. `工具 → 开发板 → 开发板管理器` 搜索并安装 **"Arduino ESP32 Boards"** (Arduino 官方版, 不是 Espressif 那个)。
3. 选板: `工具 → 开发板 → Arduino ESP32 Boards → Arduino Nano ESP32`。
4. 端口: 插上 USB-C 线后选对应 `/dev/cu.usbmodemXXXX` (Mac) 或 `COMx` (Win)。

### Step 2: 安装库 (库管理器)

| 库名 | 作者 | 说明 |
|:--|:--|:--|
| `Adafruit SSD1306` | Adafruit | OLED 驱动 |
| `Adafruit GFX Library` | Adafruit | 图形基础 |
| `Adafruit Fingerprint Sensor Library` | Adafruit | 指纹协议 |

> ESP32 自带 `Wire`, 不需安装; **不要安装 SoftwareSerial** (本代码已弃用)。

### Step 3: 先单独录制 ISD1820 语音

不要插 Arduino, 单独给 ISD1820 接 5V/GND/喇叭:

1. 按住 **REC**, 对麦克风说 **"欢迎回家"** (≤10 秒)。
2. 松开 REC。
3. 按 **PLAYE** 验证播放正常。

> ISD1820 只能存 1 段, 推荐录"欢迎回家", 失败提示由蜂鸣器旋律负责。

### Step 4: 按接线表组装

建议顺序 (先低风险后高风险):

1. **OLED** → 上电后能看到开机动画 (扫描线 + 锁图标 + 星点)。
2. **蜂鸣器** → 听到开机旋律 C-E-G-C-E。
3. **指纹模块** → 串口窗口打印 `指纹模块: OK`。
4. **KY-038 + MAX9812** → 拍手能触发"声音唤醒"。
5. **RGB LED** → 待机时红色呼吸, 唤醒时蓝色脉冲。
6. **继电器** → 测试匹配后听到"咔哒"吸合。
7. **ISD1820** → 解锁后播报"欢迎回家"。

### Step 5: 上传代码

打开 `fingerprint_door_lock.ino` → `上传`。
**上传完成后必须按一次 RESET**, 让 USB-CDC 重新枚举, 才能看见串口输出。

### Step 6: 录入指纹

打开串口监视器, **波特率 115200**:

- 发送 `e` → 进入录入模式, 按提示放手指 2 次 (中间有"移开手指"提示)。
- 发送 `d` → 清空全部指纹 (谨慎!)。
- 最多可存 127 枚。

### Step 7: 联调

1. 拍手 / 说话 → 系统从待机进入唤醒。
2. 把已录入的手指放上 → 旋律响起 + RGB 绿渐暗 + 继电器开 + 喇叭播报。
3. 按错误手指 → 抖动 X 动画 + 下行警告音。
4. 等 5 秒 → 自动上锁 + 下行旋律。
5. **彩蛋**: 连续解锁 5 次, 会播放一段"超级玛丽"风格升级音 🍄。

### Step 8: 调灵敏度

- KY-038 板载电位器: 不会自动唤醒就逆时针拧 (灵敏度↑); 误触发就顺时针。
- 代码 `detectSound()` 中 `peak > 100` 阈值可改 (60–200)。

## 七、OLED 动画与蜂鸣器旋律一览

| 场景 | OLED 动画 | 蜂鸣器旋律 | RGB |
|:--|:--|:--|:--|
| 开机 | 扫描线刷屏 + 锁 + 星点 + "Nano ESP32" | C4-E4-G4-C5-E5 上行 | 关 |
| 待机 | 锁 + 呼吸圆环 + 滚动 "Speak to wake up" | — | 红色呼吸 |
| 唤醒 | 麦克风 + 三圈声波扩散 | E5→C5 "叮咚" | 蓝色 |
| 等待指纹 | 旋转指纹纹路 + 双向流动进度条 | — | 蓝色脉冲 |
| 成功 | 锁打开 + 12 道放射线 + 烟花星点 + WELCOME | C5-E5-G5-C6-G5-C6 上行 | 绿色渐暗 |
| **彩蛋(每 5 次)** | 同上 | 超级玛丽风格升级音 | 绿色 |
| 失败 | 左右抖动的 X + ACCESS DENIED | 500-400-300-200-200 下行警告 | 红色 |
| 已开锁 | 锁开 + 双心 + 倒计时进度条 | — | 绿色渐暗 |
| 上锁 | 回到待机 | G4-E4-C4 平稳下行 | — |
| 超时 | 回到待机 | A4-A4 短促 | — |

## 八、常见问题

| 现象 | 原因 / 排查 |
|:--|:--|
| 串口看不到输出 | 上传后按 RESET; 监视器波特率改 **115200** |
| `指纹模块: 未找到` | TX/RX 是否交叉; 是否接 3.3V; 测电源 |
| OLED 黑屏 | I²C 地址换 0x3D; 检查 SDA/SCL 是否反 |
| 不会被声音唤醒 | KY-038 电位器调灵敏度; 阈值改 60 |
| 继电器一直响 | IN 是否高低电平相反 (有些模块低电平触发, 把代码里 HIGH/LOW 互换) |
| RGB 颜色不对 | 共阳极板把 `RGB_COMMON_ANODE` 设 `true` |
| 上传报错 `Failed to connect` | 双击 RESET 按钮进入 bootloader, 再上传 |
